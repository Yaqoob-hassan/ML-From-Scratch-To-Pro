# Day 04 — Matplotlib Topics Covered

## Topics

1. **Importing Matplotlib & NumPy** — Setting up the core libraries for plotting
2. **Basic Line Plot** — Plotting x/y arrays using `plt.plot()` and `plt.show()`
3. **Understanding Figure & Axes** — `plt.figure()`, `plt.subplots()`, and subplot mosaic layouts
4. **Drawing a Plot with Axes Object** — Using `fig, ax = plt.subplots()` and `ax.plot()`
5. **Adding Multiple Subplots** — `fig.add_subplot()` to place multiple axes in one figure
6. **Supertitle & Figure-level Labels** — `plt.suptitle()` for figure-wide titles
7. **Line Styling** — Changing line appearance with `linestyle` (`'dashed'`, `'dotted'`)
8. **Labels & Font Properties** — `plt.xlabel()`, `plt.ylabel()`, `fontsize`, `weight`
9. **Adding a Grid** — `plt.grid()` with custom color, linestyle, and axis options
10. **Advanced Subplot Types** — Histogram (`ax.hist`), Scatter (`ax.scatter`), Line in subplots
11. **Sample Graph Workflow** — Combining `np.linspace`, multiple plots, legend, title, and labels in one clean workflow
